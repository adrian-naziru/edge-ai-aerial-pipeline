# End-to-end-Edge-AI-Pipeline-for-Vision-Tasks-on-Aerial-and-Satellite-Imagery
End-to-end edge AI pipeline for semantic segmentation and object detection on aerial imagery. Lightweight Mask2Former (1.16M params, mIoU 0.786) + YOLOv8n + YOLO7-ITCVD deployed via ONNX Runtime on NXP i.MX 8M Plus EVK. Fully offline, no cloud dependency.
